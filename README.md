# ogu_scan

ブラウザで動作するドキュメントスキャン & PDF 共有アプリ。
インストール不要・サーバーレス・社外秘書類をクラウドに送信しない。

## 特徴

- カメラで書類を映すと自動検出・自動撮影
- 台形補正（パースペクティブ変換）で斜め撮りも正面に補正
- フィルター：オリジナル / 文書強調 / 白黒
- PDF 生成 → iOS ネイティブ共有シートから Teams・OneDrive・メールへ直接送信
- 全処理をブラウザ内で完結（画像データの外部送信なし）

## 対応環境

| ブラウザ | バージョン |
|---|---|
| Safari (iOS) | 15 以上 |
| Chrome | 90 以上 |
| Edge | 90 以上 |
| Firefox | 88 以上 |

HTTPS 環境必須（カメラ API 要件）

## 使い方

1. URL をブラウザで開く
2. カメラを書類に向ける
3. 緑の枠が安定したら自動撮影（シャッターボタンで手動撮影も可）
4. 複数ページある場合は繰り返す
5. `↑` ボタン → 共有シートから Teams / OneDrive / メール を選択

## フィルター切替

左下の `AUTO` ボタンでサイクル切替。

| 表示 | 効果 |
|---|---|
| AUTO | オリジナル（そのまま） |
| DOC | 文書強調（白飛ばし・墨入れ） |
| B&W | 白黒二値化 |

## ローカル開発

```bash
cd ~/Projects/ogu_scan
npx serve .
# http://localhost:3000 で起動
# ※ カメラは HTTPS または localhost のみ動作
```

## デプロイ（Railway）

1. GitHub にリポジトリを push
2. Railway ダッシュボード → New Project → Deploy from GitHub Repo
3. リポジトリを選択 → 自動デプロイ完了
4. URL を社内メンバーに共有するだけで即利用可能

## 技術スタック

| 項目 | 内容 |
|---|---|
| 実行環境 | ブラウザ完結（Node.js 不要） |
| カメラ | MediaDevices.getUserMedia (WebRTC) |
| 画像処理 | Canvas 2D API |
| 台形補正 | DLT ホモグラフィ + 双線形補間 |
| PDF 生成 | jsPDF v2.5.1 |
| 共有 | Web Share API (iOS ネイティブ共有シート) |
| ホスティング | Railway（静的ファイル配信） |
